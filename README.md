# Benim Web Sitem

Bu proje kişisel hobilerimi ve yapılacaklar listemi yönetmek için geliştirdiğim basit bir web uygulamasıdır.  
HTML, CSS ve JavaScript kullanılarak yapılmıştır.

## Özellikler
- Hobi ekleme
- Silme ve düzenleme
- Tamamlananları işaretleme
- LocalStorage ile verileri saklama
